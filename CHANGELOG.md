# Changelog

## [0.1.0] - unreleased

### Added
- Project skeleton with full repo (premake5 / packman / repoman) setup.
- `cable.sim` extension: procedural cable creation via `CableSpec` + `build_cable`.
- Volume deformable cable with auto hex simulation mesh.
- Deformable physics material (Young's modulus, Poisson's ratio, density).
- `Start` (kinematic) / `End` rigid-body anchors with auto attachments.
- `CreateCable` Kit command (Undo/Redo).
